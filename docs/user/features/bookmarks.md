# Bookmarks

Save important messages for quick access later.

---

## What Are Bookmarks?

Bookmarks are your personal collection of saved messages. Use them to:

- **Remember** important information
- **Track** action items and decisions
- **Reference** useful content quickly
- **Organise** things you want to revisit

Your bookmarks are **private** ‚Äî only you can see what you've saved.

---

## Saving Bookmarks

### Bookmark a Message

1. Hover over the message you want to save
2. Click the **bookmark** icon (üîñ)
3. The message is saved to your bookmarks

### Visual Confirmation

- The bookmark icon fills in when saved
- A brief notification confirms the save

---

## Viewing Bookmarks

### Access Your Bookmarks

1. Click the **menu** icon in the header
2. Select **Bookmarks**
3. Browse your saved messages

### Bookmark List

Your bookmarks show:
- Message content
- Who sent it
- Which channel
- When it was posted
- When you bookmarked it

---

## Managing Bookmarks

### Removing a Bookmark

**From the original message:**
1. Find the message in the channel
2. Click the filled bookmark icon
3. The bookmark is removed

**From your bookmarks list:**
1. Open your bookmarks
2. Click the **remove** icon on the bookmark
3. Confirm removal

### Organising Bookmarks

Bookmarks are listed chronologically (newest first). Use search within bookmarks to find specific saved items.

## Bookmarks vs Pins

| Bookmarks | Pins |
|-----------|------|
| Personal ‚Äî only you see them | Public ‚Äî everyone in channel sees them |
| Save any message you can see | Only admins can pin |
| For your own reference | For channel-wide important info |
| Unlimited | Limited per channel |

## Troubleshooting

<details>
<summary><strong>I can't find the bookmark icon</strong></summary>

The bookmark icon appears when you hover over a message. Make sure you're:
- Hovering over the message
- Looking at the action icons that appear
- Using a supported browser

</details>

<details>
<summary><strong>My bookmark disappeared</strong></summary>

Possible reasons:
- The original message was deleted
- You accidentally removed the bookmark
- The channel was deleted or you lost access

Bookmarks are tied to the original message ‚Äî if it's deleted, the bookmark goes too.

</details>

<details>
<summary><strong>Can I export my bookmarks?</strong></summary>

Currently, bookmarks stay within the platform. For important information you need elsewhere, copy the content manually.

</details>

---

## Privacy

- **Your bookmarks are private** ‚Äî no one else can see them
- **Channel rules still apply** ‚Äî you can only bookmark messages you can view
- **Deleted messages** ‚Äî if the original is deleted, the bookmark is removed
