# Customisation

Personalise your experience to match your preferences.

---

## Profile Customisation

### Display Name

Your display name is what others see:

1. Go to your **Profile**
2. Click **Edit Profile**
3. Change your **Display name**
4. Save changes

Tips:
- Use your real name or preferred nickname
- Keep it recognisable to your community
- Avoid impersonating others

### Profile Picture

Add a photo or avatar:

1. Go to your **Profile**
2. Click your current picture (or the placeholder)
3. Upload a new image
4. Adjust cropping if needed
5. Save

Recommended:
- Square images work best
- At least 200x200 pixels
- Clear face or distinctive image

### About / Bio

Add a brief description:

1. Go to your **Profile**
2. Click **Edit Profile**
3. Fill in the **About** section
4. Save

Ideas for your bio:
- A brief introduction
- Your interests
- Location (if comfortable sharing)
- What brought you to the community

## Appearance Settings

### Theme

Choose your visual preference:

| Theme | Description |
|-------|-------------|
| **Dark** | Dark background, light text (default) |
| **Light** | Light background, dark text |
| **System** | Follows your device's setting |

To change:
1. Go to **Settings > Appearance**
2. Select your preferred theme

### Font Size

Adjust text size for readability:

| Size | Best For |
|------|----------|
| **Small** | Seeing more content on screen |
| **Normal** | Default balance |
| **Large** | Easier reading |

### Compact Mode

Show more messages on screen:
- **On** — Reduced spacing between messages
- **Off** — More breathing room (default)

## Channel Organisation

### Favourites

Pin frequently used channels to the top:

1. Right-click a channel name
2. Select **Add to Favourites**
3. The channel appears in your Favourites section

Remove favourites the same way (Select **Remove from Favourites**).

### Channel Order

Within sections, channels appear in their default order. Your favourites appear separately at the top for quick access.

### Hiding Channels

If a zone has many channels:

1. Right-click a channel you rarely use
2. Select **Hide Channel**
3. The channel is hidden (but you can still search for it)

To show hidden channels:
1. Go to **Settings > Channels**
2. See your hidden channels
3. Click **Unhide** to restore

## Privacy Settings

### Online Status

Control whether others see you're online:

| Setting | Others See |
|---------|-----------|
| **Online** | Green dot when active |
| **Away** | Yellow dot after inactivity |
| **Invisible** | Appear offline (you can still use everything) |
| **Do Not Disturb** | Indicates you're busy |

To change:
1. Click your profile picture
2. Select your status
3. Or go to Settings > Privacy > Online Status

### Read Receipts

Control whether others know you've read messages:
- **On** — Senders see when you've read
- **Off** — Your reads are private

### Profile Visibility

Choose what's visible on your profile:
- **Full** — Anyone in your zones can see all details
- **Basic** — Name and picture only
- **Minimal** — Name only

## Accessibility

### Motion

If animations cause discomfort:

1. Go to **Settings > Accessibility**
2. Enable **Reduce motion**
3. Animations are minimised

### Keyboard Navigation

Navigate without a mouse:
- **Tab** — Move between elements
- **Enter** — Activate buttons/links
- **Esc** — Close dialogs
- **Arrow keys** — Navigate lists

### Screen Reader Support

The platform is designed to work with screen readers:
- Proper heading structure
- Image alt text
- ARIA labels on controls
- Landmark regions

## Keyboard Shortcuts

Customise shortcuts (where supported):

| Default | Action | Customisable |
|---------|--------|--------------|
| `Ctrl/Cmd + K` | Open search | Yes |
| `Esc` | Close dialogs | No |
| `Enter` | Send message | No |
| `Shift + Enter` | New line | No |

To customise:
1. Go to **Settings > Keyboard Shortcuts**
2. Click the shortcut you want to change
3. Press your preferred key combination
4. Save

## Resetting to Defaults

If you want to start fresh:

1. Go to **Settings**
2. Scroll to the bottom
3. Click **Reset to Defaults**
4. Choose what to reset:
   - Appearance only
   - Notifications only
   - All settings
5. Confirm

Note: This doesn't affect your account, messages, or profile — just your personal settings.

## Troubleshooting

<details>
<summary><strong>My settings aren't saving</strong></summary>

Try these:
1. Check your internet connection
2. Refresh the page
3. Try a different browser
4. Clear browser cache and try again

</details>

<details>
<summary><strong>Theme looks wrong</strong></summary>

Possible issues:
- Browser dark mode override — check browser settings
- System theme conflicts — try setting explicitly to Light or Dark
- Cache issue — clear cache and refresh

</details>

<details>
<summary><strong>My profile picture won't upload</strong></summary>

Check these:
- File size under 5MB
- Format: JPG, PNG, or GIF
- Image isn't corrupted
- Try a different image

</details>
