## Navigating Zones

If you have access to multiple zones, you can switch between them:

1. Click the **zone selector** in the header
2. Choose the zone you want to view
3. The sidebar updates to show that zone's channels

Each zone is a completely separate space with its own:
- Channels and sections
- Members
- Calendar events

## Sending Messages

### Basic Message

1. Click in the message input at the bottom
2. Type your message
3. Press **Enter** or click **Send**

### Formatting

You can format your messages:

| Format | How to Type | Result |
|--------|-------------|--------|
| Bold | `**text**` | **text** |
| Italic | `*text*` | *text* |
| Code | `` `code` `` | `code` |
| Link | Just paste URL | Clickable link |

### Mentioning People

Type `@` followed by a name to mention someone:

```
Hey @Alice, what time does the event start?
```

The mentioned person will receive a notification.

### Replying

To reply to a specific message:

1. Hover over the message
2. Click the **Reply** button
3. Type your response
4. Your reply will be linked to the original

## Using the Calendar

### Viewing Events

1. Click **Calendar** in the sidebar or navigation
2. Browse events in your chosen view:
   - **Month** ‚Äî Overview of the month
   - **Week** ‚Äî Detailed weekly view
   - **List** ‚Äî Upcoming events in order

### Event Details

Click any event to see:
- Full description
- Date, time, and location
- Who's attending
- RSVP options

### RSVPing

1. Open an event
2. Click your response:
   - **Going** ‚Äî You'll be there
   - **Maybe** ‚Äî You're interested
   - **Not Going** ‚Äî Can't make it
3. The organiser is notified

## Private Messages

### Starting a Conversation

1. Click the **messages** icon (üìß) in the header
2. Click **New Message**
3. Search for and select a recipient
4. Type and send your message

### Privacy

Private messages are **end-to-end encrypted**:
- Only you and the recipient can read them
- Not even platform administrators can access them
- They're protected by cryptographic gift-wrapping

## Tips for New Members

### Getting Started

1. **Browse first** ‚Äî Read recent messages to understand the community culture
2. **Introduce yourself** ‚Äî Post in the welcome or general channel
3. **Be yourself** ‚Äî Authentic participation builds community

### Good Practices

- **Stay on topic** ‚Äî Use the appropriate channel for your message
- **Be respectful** ‚Äî Remember there's a person behind every username
- **Use reactions** ‚Äî They're a low-noise way to participate
- **Ask questions** ‚Äî The community is here to help

### Common Mistakes to Avoid

- Posting the same message in multiple channels
- Using all caps (it reads as shouting)
- Sharing personal information publicly
- Ignoring channel topics
